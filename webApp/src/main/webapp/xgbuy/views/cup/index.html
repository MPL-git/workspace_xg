活动说明

竞猜形式：

1、球队竞猜赢积分、竞猜最后冠军队伍赢积分

2、竞猜分为三种情况，主胜、平、客胜

3、投注积分，押对赢得更多积分

4、用户可消耗积分兑换商品

5、猜对最后冠军队伍赢积分奖励

规则说明：

1、根据真实比赛结果押注积分，下注积分为20起的任意数量，可选择多种比赛结果。投注时间截止比赛前。根据比赛结果，可获得对应的积分奖励。

2、对于每一场比赛，根据你对于比赛的（90分钟及补时）结果的判断，选择“主胜、平、客胜”，设定对应的投注积分（积分不可提现，可用于兑换积分商城奖品），确认后即完成流程。如果你猜错了比赛的结果，则无法获得奖励。如果猜中结果，奖励将以积分形式派发。返奖积分 = 设定的投注积分数量 x 下单时的实时赔率。

3、加时赛及以点球决胜负的赛果均不予计算，积分奖励在比赛结束后系统自动发放。

4、消耗积分可兑换活动页商品，个别商品每天限量更新，先到先得。

5、每场比赛开始时刻将截止竞猜投注，未截止前用户可任意投注，不设上限，一个账号可多次投注。

6、冠军竞猜模式说明：在截止日期前下注积分，投注心仪的冠军队伍，该队伍假如在最后的总决赛中获胜，下注用户将获得对应赔率的积分奖励，一个账号可以竞猜多只队伍，不设限，比赛结果以最终实际赛事结果为准。

积分获得方式：

1、醒购APP内任意订单每满100元送100积分，上不封顶，多买多送。

2、APP内完成指定积分任务获得积分。

3、绑定手机号码的新老用户，系统将赠送500积分。

温馨提示：

具体活动时间请留意平台公告。

最终活动解释权为醒购所有。
